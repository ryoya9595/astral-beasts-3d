# アストラルビースト 3D 〜はじまりの章〜（デモ版）

ブラウザで遊べるローポリ3Dモンスター RPG デモ。ゲーム開始〜最初の守人クリアまで。

▶ https://ryoya9595.github.io/astral-beasts-3d/

## 操作
- 移動: 画面内バーチャルスティック（深く倒すとダッシュ） / WASD・矢印キー（Shiftでダッシュ）
- はなす・しらべる: Aボタン / Zキー
- メニュー: ☰ / Xキー

## クレジット
- BGM: フリー音楽素材 [魔王魂](https://maou.audio/)
  - maou_game_village01 / maou_game_field05 / maou_game_battle32
- 企画・制作: アストラル開発室
